2018阅读计划
Created 2018-01-19| Updated 2020-04-22
Post View:

计划不裱出来难以执行
从小到大列过无数的计划，几乎无一完全实现，反思了很久发现是没有公诸于众，于是有了本文 。书库里书是很多，准备挑出十余本作为今年的阅读计划。
闲书我就不列了，茶余饭后自然会找时间读。

一月

一月已过去小半，这个月我已经在研读的是 《web开发权威指南》 ，这本书本身是有些导读性质的，并不像什么XX大全之类的面面俱到，让人只能当当工具书用，也不像30天精通XX系列，读完事实上并不能精通（笑）。知识点涵盖到一些较新的前端技术，读完此书并不能让人成为前端大师，但至少给了 把钥匙。

二月

作为前书的补充，把 《HTML5+CSS3》 浏览一下，这本是可以当工具书用的，但本身我工作的职责不在前端这一块，所以躺在书柜里好久了。 当然这种粗略的翻阅并不能算到阅读计划中 ，二月还需要再找一本 nodejs 相关入门书籍，满足日常开发实验所需。

三月

作为Java程序员，重读 《Thinking in Java》 不解释。

四月

理由同上，这个月需要读完 《Java程序员修炼之道》 ，本书主要讲解Java7中的一些特点和新特性，以及其他JVM语言与Java混合开发。现在Scala、Kotlin这么火热，总要一技傍身吧。

五月

《Java技术手册》 这本书是衔接7和8的，提及到两个版本的新特性，内容主要在于演示如何充分利用现代API和开发过程中的最佳实践，属于实用派书籍。

六月

六月，请对我好一点。那么我就选择 《Java 8实战》 好了。说来惭愧，大家都在研究Java9了 ，但由于现在公司项目开发的主要版本还在6和7,所以对8的新特性和 函数式编程 只是耳闻，心生向往已久。技能数新增一枝总让人兴奋不是吗！

七月

作为Java本年度阅读计划的终结（我也觉得有点多了），这个月献给 《函数式编程思维》 ，作为8实战的理论实践补充。

八月

Java 结束，轮到 Js 了。八月准备读的是 《同构JavaScript应用开发》 ,主要关注点在构建和维护Js应用，同时接触一下 AngularJs。

九月 十月

比较难取舍，但是这两个月要完成 《JavaScript编程精粹》&&《学习JavaScript数据结构与算法》 的学习与阅读。虽然一直都在用Js写代码，但是好多点都是知其然，需要好好花心思阅读学习。

十一月

Python 入门书籍跪求推荐

十二月

当然是排给热门的 《流畅的Python》 了。讲道理 Python 是真的好用，用来写过脚本，比起Java是真的优美太多了(当然主要是由于我Java也是刚起步，半桶水晃荡)。

备选书籍

尤有余力，还有 《程序员的趣味算法题》||《七周七语言》||《全端Web开发》||《Spring Boot实战》 作为补充。

写在最后

跪求 nodeJs和Python2 3 的入门书籍，稍微有些难度没关系，但是最好有些趣味性，本人最痛恨教纲类的书籍，拜谢！
